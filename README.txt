ArmyZon (사회의 아마존을 대체하는 군 내부의 아미존) by 최유성

개요
군 마트의 현재 재고 현황과 물품 신청과 물품 목록을 볼 수 있는 어플리케이션이다.

개발 배경
전국 각지의 많은 장병들이 각 부대 내부의 영내 마트를 이용하고있다. 
장병들은 고된 훈련과 지친 하루 일과를 맛있는 음식으로 달래며 행복함을 느낀다.
하지만 막상 마트에 들어가면 원하는 물품이 없는 경우가 많았다.
그래서 현재 우리부대 마트의 현 재고 현황과 수요하는 물품들을 신청할 수 있는 앱의 필요성을 느끼게 되었다.

기능
1. 현재 군 마트 내부의 물품 재고 현황 확인 기능 
2. 개인이 신청하고자하는 물품을 신청하는 기능(미구현)
3. 신청된 물품 리스트를 보여주는 기능(미구현)
4. 공지 사항

활용 방안
- 군 마트까지 고생해서 갔는데 원하는 물품을 구입하지 못하는 불상사를 방지할 수 있음
- 원하는 물품을 신청할 수 있음
- 추후에 업데이트를 통해서 오늘,이번주,이번달에 가장 많이 팔린 랭킹 TOP5등을 추려서 홍보할 수 있음
- APP을 이용하는 장병들에게 할인 쿠폰과 같은 이벤트 행사를 진행할 수 있음


개발하면서...
지금까지 스스로 혼자 어플을 개발해본적이 없었는데 4박5일간의 짧은 시간이지만 국방부 오픈소스 아카데미를 통해 이런 좋은 경험을 할 수 있어서 정말 좋았습니다. 
열정적으로 강연해주신 손영수 교수님과 진도를 잘 따라갈 수 있도록 일대일로 많은 시간을 할애해주신 차용빈 조교님께 감사합니다.
사실 완벽하지도 완성된 어플리케이션도 아니지만 결과물을 남기고 싶어서 커밋하게되었습니다. 
지난 시간동안 정말 고생하셨습니다 . ^^~
